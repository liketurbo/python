# Operators

| Operator | Name                              | Example          |
| -------- | --------------------------------- | ---------------- |
| `*`      | Multiplication                    | `3 * 5 => 15`    |
| `**`     | Exponent                          | `2 ** 3 => 8`    |
| `/`      | Division                          | `22 / 8 => 2.75` |
| `//`     | Integer division/floored quotient | `22 // 8 => 2`   |
